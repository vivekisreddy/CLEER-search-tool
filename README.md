## **CLEER - Engineering Education Research Search Tool**  

### 🚀 **Overview**  
CLEER (**Collaborative Learning in Engineering Education Research**) is a modern, AI-powered search tool designed to help **PhD students, educators, and pedagogical advisors** quickly find and analyze research papers in engineering education.  

This is an **initial UI prototype** focused on **clean design, intuitive navigation, and user experience** using **React + Vite + Material UI**. The backend is currently a placeholder using **Node.js**, with no database integration yet.  

---

### 🎨 **Features (Prototype Stage)**  
✅ **Minimalist Google Scholar-style UI**  
✅ **AI-assisted search bar with keyword suggestions**  
✅ **Advanced filtering (Technical vs. Theoretical, Impact across Fields, etc.)**  
✅ **Trust indicators (Author reputation, Journal ranking, etc.)**  
✅ **Notepad for idea generation & research brainstorming**  
✅ **User-specific customization for different researcher types**  
✅ **Dark mode support & mobile-friendly UI**  
✅ **Navigation bar with Home, Saved Papers, Community, and Settings**  

---

### 🛠 **Tech Stack**  
- **Frontend:** React + Vite + Material UI  
- **Backend:** Node.js (Temporary API with no database)  
- **Version Control:** Git & GitHub  

---

### 📌 **Setup Instructions**  
1️⃣ **Clone the Repository**  
```sh
git clone https://github.com/vivekisreddy/CLEER-search-tool.git
cd CLEER-search-tool
```
2️⃣ **Install Dependencies**  
```sh
npm install
```
3️⃣ **Run the Development Server**  
```sh
npm run dev
```
4️⃣ **Open in Browser:**  
Vite will display a local development server (e.g., `http://localhost:5173/`).
